HTML Tables

Cheatsheet

HTML tables allow web developers to arrange data into rows and columns.

HTML tables are created using the <table> tag in which the tag is used to create table rows and <td> tag is used to create data cells. The elements under <td> are regular and left aligned by default

Elements & Attributes HTML Description
Table <table> Defines the table and will contain rows and columns
Table Head <thead> Defines the headings of table columns
Table Body <tbody> A semantic element that contains all the table data not contained in the table's heading and footer.
Table Footer <tfoot> Defines the footer of the table and uses table rows for summarising content.
Table Row <tr> Used to add rows to a table before adding the data and headings
Table Heading <th> Adds titles to rows and columns, and must be enclosed in a table row <tr> element
Table Data <td> Defines individuals data cells of a table and must be nested in a table row <tr> element
rowspan Attribute <rowspan> Defines the number of rows that a cell can span across the table when applied to a th or td element. The value is 1 by default and can be any positive integer.
colspan Attribute <colspan> Defines the number of columns that a cell can span across the table when applied to a th or td element. The value is 1 by default and can be any positive integer
